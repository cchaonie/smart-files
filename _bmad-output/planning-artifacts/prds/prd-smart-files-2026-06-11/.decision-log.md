# Decision Log — Smart Files → Home NAS

## 2026-06-11

- PRD created, status: draft. Initial Discovery phase started.
- **Vision**: Family photo hub — upload from Android phones, AI-tag photos, browse timeline, share with family members.
- **User journeys confirmed**: UJ-1 (phone cleanup), UJ-2 (baby photo search), UJ-3 (family multi-user).
- **Working mode**: Coaching path, journey-led entry point.
- **Upload behavior**: Auto-remind (not auto-upload).
- **AI classification**: All types confirmed — tags, face recognition deferred to post-MVP, time-based grouping. Manual albums sufficient for v1.
- **Target platform**: Android only for v1 (iOS deferred).
- **Multi-user**: Personal accounts with personal libraries + opt-in album sharing.
- **Storage**: Pooled external USB drives (mergerfs or equivalent); folder structure `{user}/{date}`; unlimited per-user with visible remaining space.
- **Search**: Tag-based only for v1 (no semantic/CLIP search).
- **External sharing**: Deferred to post-MVP.
- **Remote access**: Cloudflare Tunnel already set up.
- **NFRs confirmed**: HTTPS, password strength, photo privacy, upload throughput, thumbnail loading, storage architecture, upload resilience, pool resilience.
